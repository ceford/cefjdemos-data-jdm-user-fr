## Objectif

Le module personnalisé vous permet de créer du contenu de module en utilisant un éditeur WYSIWYG. Il est idéal pour des messages simples, éventuellement avec des images.  

## Créer le module personnalisé

Utilisez l’une des méthodes suivantes :
* Sélectionnez l’élément **Modules** à partir du tableau de bord d'accueil. Ou...
* Sélectionnez **Contenu / Modules du site** dans le menu de l’administrateur.
* Sélectionnez l’élément **Personnalisé** dans la liste des Modules (Site).

Cela créera le nouveau module et ouvrira le module pour la configuration.

## Configuration du module

Pour une utilisation standard du module, il y a quelques paramètres :

- **Titre** Entrez un nom pour le module.
- **Contenu du module** Tapez le contenu comme vous le feriez pour un article.
- **Afficher/Masquer le titre** Choisissez si vous souhaitez afficher le titre ou non en basculant sur *Afficher* ou *masquer*.
- **Position** Définissez une position où vous souhaitez afficher le module sur le front-end. Les positions sont définies par votre modèle. Cet exemple utilise la position *Sidebar Right* du modèle Cassiopeia de Joomla.
- **Statut** Par défaut, le statut du module est **Publié**. D'autres options sont **Non publié** et **Supprimé**.
- **Début de publication** Vous pouvez programmer le début de la publication du module.
- **Fin de publication** Vous pouvez programmer la fin de la publication du module.
- **Accès** Si vous souhaitez contrôler qui peut voir le module sur le frontend, vous pouvez choisir un niveau d'accès.
- **Ordre** Utilisé pour contrôler où le module s'affiche dans la position que vous avez sélectionnée. Par exemple, vous pouvez avoir plusieurs modules dans la barre latérale et vouloir que celui-ci soit en haut ou en bas, ou quelque part entre d'autres dans la barre latérale. Cela peut être un peu déroutant au début car le champ affiche une position numérotée et un nom de module. Le nom peut être un module qui n'est pas publié. Il est important de se rappeler que le nombre le plus bas sera en haut et le nombre le plus élevé en bas.
- **Note** Il peut être utile d'utiliser le champ de notes si, par exemple, vous utilisez le même type de module à plusieurs endroits.

### Onglet d'affectation du menu

Par défaut, le module sera publié **Sur toutes les pages**.

Vous pouvez également choisir via une liste déroulante **Aucune page**, **Uniquement sur les pages sélectionnées** ou **Sur toutes les pages sauf celles sélectionnées**. Les deux dernières options vous fournissent un arbre de menus des menus utilisés sur le site Web où vous pouvez sélectionner/désélectionner des pages.

### Autre

L'onglet **Options** vous permet de définir une image de fond pour le module.

L'onglet **Avancé** contient des paramètres liés à la disposition du module lorsqu'il est sorti.

L'onglet **Permissions** vous permet de contrôler ce que les groupes d'utilisateurs peuvent faire avec le module.

## Publier le Module

Quand vous êtes prêt, sélectionnez le bouton **Enregistrer & Fermer**.

Le module sera publié dans la barre latérale du site web et affichera
le texte que vous avez saisi.
